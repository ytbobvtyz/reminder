# Файлы проекта MCP Scheduler Agent

## Основные файлы:

### 1. mcp_server.py
**Описание:** MCP сервер с планировщиком задач
**Содержание:**
- Модель ReminderMCP с инструментами: add_reminder, list_tasks, get_due_tasks, mark_done
- APScheduler с SQLite persistence
- SQLite база данных для хранения задач
- Модель Server для обработки MCP запросов
**Статус:** ✅ Готово

### 2. agent.py
**Описание:** LLM агент для обработки запросов пользователя
**Содержание:**
- Класс ReminderAgent с подключением к OpenRouter API
- Анализ естественного языка для определения напоминаний
- Подключение к MCP серверу через stdio_client
- Синхронная обертка SyncReminderAgent для Streamlit
**Статус:** ✅ Готово (исправлена версия OpenAI)

### 3. app.py
**Описание:** Streamlit UI с чат-интерфейсом
**Содержание:**
- Чат с историей сообщений
- Фоновый polling для уведомлений (каждую секунду)
- Toast уведомления при выполнении задач
- Сайдбар с активными задачами и статистикой
- Управление подключением к агенту
**Статус:** ✅ Готово

## Зависимости проекта:

### Python библиотеки:
- **streamlit** - для UI интерфейса
- **mcp** - для MCP сервера и клиента

### API сервисы:
- **OpenRouter API** - для LLM модели Qwen 3.6 Plus Preview

## Потенциальные проблемы и решения:

### 1. Версия OpenAI
**Проблема:** Ошибка `Client.__init__() got an unexpected keyword argument 'proxies'`
**Решение:** Использовать openai==0.28.1 вместо openai==1.54.0

### 2. Подключение к MCP серверу
**Проблема:** Не запускается MCP сервер как дочерний процесс
**Решение:** Проверить наличие всех зависимостей и права на выполнение файлов

### 3. API ключ OpenRouter
**Проблема:** API ключ не работает или не найден
**Решение:** Проверить файл .env и правильность ключа

### 4. Streamlit UI
**Проблема:** Уведомления не появляются
**Решение:** Проверить работу фонового polling и toast уведомлений

## Инструкция по запуску:

### Быстрый запуск:
```bash
# Установка зависимостей
pip install -r requirements.txt

# Проверка проекта
python simple_test.py

# Тестирование агента
python test_agent.py

# Запуск приложения
streamlit run app.py
```

### Альтернативный запуск:
```bash
# Использование скрипта
./start.sh

# Или
bash start.sh
```
