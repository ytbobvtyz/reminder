4.2 agent.py — LLM агент

Ключевые методы:

Метод	Что делает

System prompt:

Фоновый polling (ключевая механика для уведомлений):

python
# В отдельном потоке или через st.empty() + time.sleep()
def check_notifications():
    while True:
        due_tasks = agent.call_mcp_tool("get_due_tasks", {})
        for task in due_tasks:
            st.toast(f"🔔 {task['message']}", icon="⏰")
            agent.call_mcp_tool("mark_done", {"task_id": task["id"]})
        time.sleep(1)  # проверка каждую секунду

5. Потоки данных
5.1 Создание напоминания
text
Пользователь → Streamlit (ввод текста)
    → agent.process("напомни через 30 сек попить кофе")
    → LLM анализирует: intent=reminder, message="попить кофе", delay=30
    → LLM вызывает tool: add_reminder(message="попить кофе", delay_seconds=30)
    → agent вызывает MCP сервер
    → MCP сервер: SQLite INSERT + APScheduler.add_job()
    → MCP сервер возвращает task_id
    → LLM формирует ответ: "✅ Напоминание установлено"
    → Streamlit отображает ответ в чате
5.2 Уведомление
text
APScheduler (через delay_seconds) → вызывает callback
    → MCP сервер обновляет status='done' в SQLite
    → (ничего не пушит, просто ждёт polling)

Streamlit polling (каждую секунду) → вызывает get_due_tasks
    → MCP сервер возвращает задачи с status='pending' и run_at <= now()
    → Streamlit показывает st.toast()
    → Streamlit вызывает mark_done
6. Для агента используй openrouter api модель - MiniMax M2.5. - ключ хранится в переменной OPENROUTER_API_KEY в файле .env

7. Критерии приёмки
Пользователь пишет "напомни через X секунд/минут Y"

Агент сам извлекает параметры и вызывает add_reminder

Агент отвечает с подтверждением

Через X секунд появляется всплывающее уведомление (st.toast)

Уведомление появляется без кнопки обновления

Всё работает в одном терминале: streamlit run app.py

8. Заметки для код-ассистентов
Для mcp_server.py:
Используй BackgroundScheduler (не async)

SQLite: таблицы создаются при старте

При add_reminder: run_at = datetime.now() + timedelta(seconds=delay)

Job ID = task_id (чтобы можно было отменить)

get_due_tasks: WHERE status='pending' AND run_at <= datetime('now')

Для agent.py:
Используй stdio_client для запуска MCP сервера

Получи инструменты через list_tools()

Конвертируй их в формат OpenAI

System prompt должен быть чётким: "ОБЯЗАТЕЛЬНО используй add_reminder"

Для app.py:
st.chat_input для ввода

st.chat_message для отображения диалога

Фоновый поток: threading.Thread(target=check_notifications, daemon=True)

В потоке: while True: time.sleep(1); ... st.toast(...)
